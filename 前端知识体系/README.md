## 🎯 学习建议

### 1️⃣ 循序渐进

不要跳跃式学习，先打好基础再学框架。

### 2️⃣ 实践为主

每学一个知识点，都要动手实现一个小 Demo。

### 3️⃣ 理解原理

不要只会用 API，要理解背后的原理。

### 4️⃣ 输出总结

写博客、做分享，教学相长。

### 5️⃣ 关注前沿

定期阅读技术文章，了解最新技术动态。
